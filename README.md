# visualization-project

> 项目初始框架使用 vue-cli 搭建；
> 基于 DataV 框架，整合了 Echarts、elementUI 的初始化数据大屏项目。

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
